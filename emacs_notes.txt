### Day to day usage

C-c ?   (jedi) show doc for object under point
C-c .   (jedi) goto-definition
C-c , 	(jedi) goto-definition pop marker (i.e. go back)
C-i     (helm) find code object (semantic)
S-tab	(helm-files) one level up
~/	(helm-files) to home
/	(helm-files) to root
S-tab	(helm-files) one level up
C-c C-z (python-mode) Switch to python shell
C-c C-c (python-mode) Execute buffer
C-c p f (projectile) projectile find file

Run e.g. pytest
M-x compile
 (and change the command e.g. to py.test)
